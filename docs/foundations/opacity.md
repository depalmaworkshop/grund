# Opacity

Transparency levels for consistent visual hierarchy.

## Scale

| Token | Value | Use Case |
|-------|-------|----------|
| 0 | 0% | Invisible |
| 5 | 5% | Subtle overlay |
| 10 | 10% | Light tint |
| 20 | 20% | Hover states |
| 25 | 25% | Disabled states |
| 50 | 50% | Semi-transparent |
| 75 | 75% | Heavy overlay |
| 90 | 90% | Near-opaque |
| 100 | 100% | Fully opaque |
