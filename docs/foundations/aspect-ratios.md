# Aspect Ratios

Standard ratios for images, video, and media containers.

## Ratios

| Name | Ratio | Use Case |
|------|-------|----------|
| Square | 1:1 | Avatars, thumbnails, Instagram |
| Portrait | 4:5 | Instagram portrait, product cards |
| Portrait Tall | 3:4 | Traditional portrait |
| Landscape | 5:4 | Landscape images |
| Landscape Wide | 4:3 | Traditional landscape, photos |
| Widescreen | 16:9 | Video, hero images |
| Vertical Video | 9:16 | Stories, TikTok, Reels |
| Panorama | 21:9 | Cinematic, ultra-wide |
